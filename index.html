📐 核心公式速查
Precision / Recall / F1›
Precision = TP/(TP+FP) 預測為正中有多少真的是正
Recall = TP/(TP+FN) 真正的正有多少被找出來
F1 = 2×P×R/(P+R)

醫療/詐欺：Recall 優先（不能漏）
垃圾郵件：Precision 優先（不能誤刪）
L1 vs L2 正則化›
L1 Lasso：部分權重歸零 → 稀疏、特徵選擇
L2 Ridge：所有權重縮小但不為零 → 通用防過擬合

記憶：L1 = Lasso = Leave some zero
過擬合 vs 欠擬合›
過擬合：訓練低＋驗證高 → L1/L2、Dropout、更多資料
欠擬合：兩者都高且接近 → 增加模型複雜度

學習曲線兩條線都高且接近 = 欠擬合
何時需要標準化›
需要：KNN、SVM(RBF)、PCA、神經網路、邏輯斯迴歸
不需要：決策樹、隨機森林、XGBoost

記憶：距離型/梯度型 → 需要；樹型 → 不需要
F1 計算範例›
TP=80, FP=20, FN=40
P = 80/100 = 0.80
R = 80/120 = 0.667
F1 = 2×0.80×0.667/(0.80+0.667) = 0.727
Sklearn Data Leakage 避免›
錯誤：對全部資料 fit_transform 再做 cross_val_score
正確：用 Pipeline，每個 Fold 只對訓練集 fit

X_test 只能 transform()，不能 fit_transform()！
